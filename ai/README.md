# AI Operating Layer

This folder is for AI agents, skills, prompts, context, and governance.

The main `docs/` folder teaches people. This `ai/` folder gives agents and AI-assisted workflows a clearer place to find operating context.

---

## Organizing Principle

```text
docs/       teaches people
ai/         guides agents and AI workflows
templates/  provides reusable starting points
examples/   proves usage
.github/    enforces behavior
```

---

## Sections

| Folder | Purpose |
|---|---|
| `agents/` | Agent patterns, boundaries, and safe operating rules |
| `skills/` | Reusable skill guidance and skill templates |
| `prompts/` | Prompt patterns and reusable prompt guidance |
| `context/` | Repo-level context files for AI tools and coding agents |
| `governance/` | Safety, tool intake, MCP, and model governance references |

---
