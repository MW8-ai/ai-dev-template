# AI Context

This folder is the navigation layer for repo-level AI context.

AI context files tell coding agents how to behave in this repository: what to read first, what rules matter, what tools are allowed, and what actions require human approval.

---

## Current Context Files

| File | Purpose |
|---|---|
| [`../../AGENTS.md`](../../AGENTS.md) | General agent rules and repository boundaries |
| [`../../CLAUDE.md`](../../CLAUDE.md) | Claude Code-specific guidance |
| [`../../TECH_STACK.md`](../../TECH_STACK.md) | Technology choices and rationale |
| [`../../DESIGN.md`](../../DESIGN.md) | Architecture and design decisions |
| [`../../SECURITY.md`](../../SECURITY.md) | Security policy and responsible disclosure |

---

## Context Rule

> AI context should guide behavior, not hide decisions. Humans should be able to read and understand every rule agents are asked to follow.
