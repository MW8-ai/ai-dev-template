# GROK.md

## Purpose

Guidance for Grok/xAI usage.

## Good Uses

- brainstorming
- critique
- naming
- second-opinion review
- plain-English summaries

## Caution

Do not use as sole authority for:

- production deployment
- security decisions
- database changes
- government compliance
- legal/regulatory claims

## Snapshot Rule

Capabilities, pricing, and API access change quickly. Record dated findings in `docs/07_snapshots/`.
