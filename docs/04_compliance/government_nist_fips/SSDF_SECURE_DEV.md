# SSDF_SECURE_DEV.md

## Purpose

Secure Software Development Framework-inspired checklist.

## Prepare

- define roles
- define toolchain
- define coding standards
- define AI tool rules
- define approved dependencies

## Protect

- protect code repository
- protect secrets
- protect build pipelines
- use branch protection
- use least privilege

## Produce

- review code
- run tests
- scan dependencies
- document changes
- generate reproducible builds when possible

## Respond

- track vulnerabilities
- patch dependencies
- document incidents
- improve controls after lessons learned
