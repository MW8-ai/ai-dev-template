Example neutral web app.

## Name

Example: PantryPilot

## Purpose

A small web app that helps users plan pantry-friendly meals from ingredients they already have.

## Core Loop

```text
Enter ingredients → suggest options → save favorites → refine next time
```

## Non-Goals

- no medical advice
- no paid API dependency in prototype
- no user accounts in first version
