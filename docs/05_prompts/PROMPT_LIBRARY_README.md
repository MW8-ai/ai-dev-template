# PROMPT_LIBRARY_README.md

## Purpose

Reusable prompts for AI-assisted development.

Prompts are useful, but they are secondary to:

1. repo structure
2. permissions
3. tests
4. review
5. deployment controls

## Prompt Categories

- starter prompts
- planning prompts
- builder prompts
- tester prompts
- reviewer prompts
- documentation prompts
- compliance prompts
- research prompts

## Prompt Rule

Prompts should reference repo files instead of repeating everything.

Bad:
> Here are 2,000 words of rules...

Good:
> Read `AGENTS.md`, use context level 2, and complete this scoped task.
