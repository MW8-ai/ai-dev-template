## AWS

Official cert portal: **https://aws.amazon.com/certification/**

AWS added two new AI/ML certifications in 2024. Exams cost $150–$300 USD. Valid for 3 years.

| Cert | Level | Cost | Status | Link |
|---|---|---|---|---|
| AWS Certified AI Practitioner | Foundational | ~$150 | ✅ New 2024 | https://aws.amazon.com/certification/certified-ai-practitioner/ |
| AWS Certified Machine Learning Engineer – Associate | Associate | ~$150 | ✅ New 2024 | https://aws.amazon.com/certification/certified-machine-learning-engineer-associate/ |
| AWS Certified Machine Learning – Specialty | Specialty | ~$300 | ✅ Active | https://aws.amazon.com/certification/certified-machine-learning-specialty/ |
| AWS Certified Data Engineer – Associate | Associate | ~$150 | ✅ Active | https://aws.amazon.com/certification/certified-data-engineer-associate/ |

> The AI Practitioner and ML Engineer Associate are new (2024) and represent AWS's response to the GenAI certification market. If you are starting an AWS AI track, begin with AI Practitioner.

## Choosing a Path

**Starting from zero, no cloud background:**
→ Google AI Essentials (free) → Azure AI Fundamentals (AI-900) → then pick a cloud platform

**Already a developer, want AI credibility:**
→ DeepLearning.AI Generative AI with LLMs → AWS AI Practitioner or Azure AI Engineer (AI-102)

**Want to run models locally / open-source path:**
→ fast.ai Practical Deep Learning → Hugging Face Course → HuggingFace fine-tuning track

**Cloud infrastructure focus:**
→ Pick your platform (Azure/AWS/GCP), get the foundational cert, then the ML specialty

**Vendor-neutral / governance focus:**
→ CompTIA AI+ → IBM AI Engineering (Coursera)

## Keeping This Current

Certifications retire, rename, and change pricing frequently. Microsoft in particular has retired entire exam tracks and introduced Applied Skills as a new shorter-form credential format.

This doc is tracked by the Living Updates system:
- `microsoft-learn-cert-blog` source — retirement and update announcements
- `aws-training-blog` source — new cert launches and changes
- `google-cloud-blog` source — GCP cert updates

When a meaningful change is detected, a draft note appears in `docs/06-living-updates/incoming/` for review. If it affects this doc, update the relevant row's Status column.

Before registering for any exam, verify current pricing and status at the official portal — costs and availability change without notice.
