# Code of Conduct Template

Use this template to define expected behavior for contributors, reviewers, maintainers, and AI-assisted contributors.

## Expected behavior

- Be respectful and clear.
- Review ideas and code without personal attacks.
- Assume good intent, but verify technical claims.
- Escalate security, privacy, or compliance concerns quickly.

## Unacceptable behavior

- Harassment or abusive language.
- Sharing secrets or private data in issues, pull requests, prompts, or logs.
- Approving changes without review because an AI tool generated them.

## Reporting

Document the team contact or channel for reporting concerns before using this in a real project.
