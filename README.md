# AI-Native Development Repository Scaffold

Reusable documentation and workflow scaffold for AI-assisted software development.

This repo is intentionally project-neutral. Use random example names only. Do not include client, agency, customer, internal system, or production secrets.

## What This Repo Is

A reusable source-of-truth system for:

- web apps
- web prototypes
- dashboards
- browser games
- internal tools
- AI-assisted repo workflows
- agent orchestration
- government/NIST/FIPS-aware development
