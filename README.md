# AI Dev Template

A developer playbook — not an application — for building software with professional workflows and AI assistance.

## What This Repo Is

This is a structured collection of documentation, templates, and workflow guides. You clone or fork it to give any project a solid foundation: clear processes, automated checks, and AI tooling that stays in its lane.

It is not an app. There is nothing to run. The value is in the structure and the knowledge inside it.

## Core Idea

- **Repo docs are memory** — decisions, standards, and context live here, not in someone's head
- **Git is history** — every change is tracked, reviewable, and reversible
- **CI/CD (Continuous Integration / Continuous Deployment) is enforcement** — automated checks catch problems before they reach production
- **AI tools are workers, not owners** — AI generates, humans review and decide

## Folder Overview

| Folder / File | What It Contains |
|---|---|
| `docs/` | All documentation: getting started, workflows, AI tools, compliance |
| `templates/` | Copy-paste starting points for projects, workflows, and prompts |
| `examples/` | Working examples showing templates in real project structures |
| `scripts/` | Shell scripts for repo validation and checks |
| `.github/` | GitHub Actions workflows, issue forms, and PR templates |
| `standards/` | Detailed coding, review, testing, and documentation standards |
| `AGENTS.md` | Rules and boundaries for AI coding agents |
| `TECH_STACK.md` | Technology choices and rationale |
| `DESIGN.md` | Architecture and design decisions |
