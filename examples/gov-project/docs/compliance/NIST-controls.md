# NIST Controls

Map project controls to applicable National Institute of Standards and Technology guidance. This starter file is intentionally simple so teams can expand it for their environment.

## Control map

| Area | Expected evidence |
| --- | --- |
| Access control | Role assignments, least-privilege review, and approval records |
| Audit logging | Log sources, retention, and review process |
| Configuration management | Change records and pull request history |
| Incident response | Contact list and escalation process |
