# Beginner App — Hello, World! Personal Page

This is a beginner example project. It is a simple personal web page built with plain HTML and CSS —
no frameworks, no build tools, no server required. Open one file in a browser and it works.

This example is part of the [AI-Native Developer Playbook](../../README.md) and is designed to show
what a well-organized beginner project looks like.

---

## How to Run

No installation required.

1. Download or clone this folder to your computer
2. Open `index.html` in any web browser (double-click the file, or drag it into a browser window)
3. That's it — your personal page is running locally

If you want to share it with others, you can:
- **GitHub Pages (free):** Push this folder to a GitHub repository, enable Pages in Settings
- **Netlify (free):** Drag the folder to [netlify.com/drop](https://netlify.com/drop)

## What to Do Next

Once you have this page working, here are good next steps for learning:

1. **Edit the content** — open `index.html` in a text editor and change the name, bio, and button text
2. **Change the colors** — look for the CSS variables at the top of the `<style>` block and change the color values
3. **Add a new section** — try adding a section for your projects or hobbies
4. **Learn about separate CSS files** — move the `<style>` block into its own `style.css` file and link it with `<link rel="stylesheet" href="style.css">`
5. **Learn about JavaScript** — add a `<script>` block that makes the button do something when clicked
6. **Put it online** — use GitHub Pages or Netlify to share your page with the world
